# 指标卡最终调整说明

## 调整内容

### 1. 图标改为发光圆点
- 所有指标的图标从emoji改为 `●` (发光圆点)
- 圆点尺寸：20px
- 发光效果增强：drop-shadow 0 0 8px (透明度0.8)
- 悬停时圆点放大1.2倍，发光增强

### 2. 布局调整

**从**: 3列2行均匀布局
```
┌─────┬─────┬─────┐
│  1  │  2  │  3  │
└─────┴─────┴─────┘
┌─────┬─────┬─────┐
│  4  │  5  │  6  │
└─────┴─────┴─────┘
```

**改为**: 第一行2列（较宽）+ 第二行4列（较窄）
```
┌───────────┬───────────┐
│     1     │     2     │  ← 每个占2列宽度
└───────────┴───────────┘
┌─────┬─────┬─────┬─────┐
│  3  │  4  │  5  │  6  │  ← 每个占1列宽度
└─────┴─────┴─────┴─────┘
```

### 3. CSS实现

```css
.overview-panel {
  display: grid;
  grid-template-columns: repeat(4, 1fr);  /* 4列网格 */
  grid-template-rows: auto auto;          /* 2行 */
  gap: 20px;
}

/* 第一行的卡片占2列 */
.metric-card:nth-child(1),
.metric-card:nth-child(2) {
  grid-column: span 2;
}
```

### 4. 指标顺序

**第一行（基础统计）**:
1. 汇聚部门数 - 蓝色圆点
2. 部门汇聚清单表数量 - 绿色圆点

**第二行（更新率指标）**:
3. 部门数据更新率 - 橙色圆点
4. 前置仓更新率 - 紫色圆点
5. 链路完成率 - 深绿色圆点
6. 链路异常率 - 红色圆点

## 视觉效果

### 发光圆点特点
- 尺寸适中（20px），不会过大
- 强烈的发光效果，科技感十足
- 每个圆点有独特的颜色
- 悬停时圆点放大并增强发光

### 布局优势
- 第一行较宽，适合显示基础统计数据
- 第二行较窄，4个更新率指标并排显示，便于对比
- 视觉层次清晰，重要指标（部门数、表数量）更突出
