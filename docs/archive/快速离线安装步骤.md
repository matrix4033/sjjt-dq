# 快速离线安装步骤

## 📥 在有网络的机器上

```bash
# 1. 运行下载脚本
chmod +x download-py37-packages.sh
./download-py37-packages.sh

# 2. 验证（重要！）
ls offline-packages/ | grep itsdangerous
# 必须是: itsdangerous-2.0.1 (不是 2.2.0)

ls offline-packages/ | grep click
# 必须是: click-8.0.4 (不是 8.3+)

# 3. 传输到服务器
scp python-packages-py37.tar.gz user@server:/path/to/dateQuality/
```

---

## 📦 在目标服务器上

```bash
# 1. 解压
tar -xzf python-packages-py37.tar.gz

# 2. 安装
./install-offline.sh

# 3. 验证
source api/venv/bin/activate
python -c "import flask; print(flask.__version__)"  # 应显示 2.0.3
deactivate

# 4. 配置并启动
cp api/.env.example api/.env
vi api/.env
mysql -u root -p < setup-database.sql
python3 import-csv-to-mysql.py
./start-all.sh
```

---

## ⚠️ 如果遇到问题

### 下载的版本不对
```bash
# 删除旧包
rm -rf offline-packages

# 清除缓存
pip3 cache purge

# 重新下载
./download-py37-packages.sh
```

### 安装失败
```bash
# 删除虚拟环境
rm -rf api/venv

# 重新安装
./install-offline.sh
```

### 启动报错 "No module named 'flask'"
```bash
# 检查虚拟环境
ls api/venv/bin/python

# 手动测试
source api/venv/bin/activate
python -c "import flask, pymysql, flask_cors"
deactivate
```
