# 指标卡功能完整总结

## 最终实现的功能

### 1. 指标重新设计 ✅

**6个指标**：
1. 汇聚部门数（个）- 蓝色
2. 部门汇聚清单表数量（张）- 绿色
3. 部门数据更新率（%）- 橙色
4. 前置仓更新率（%）- 紫色
5. 链路完成率（%）- 深绿色
6. 链路异常率（%）- 红色

### 2. 布局优化 ✅

**第一行2列（较宽）+ 第二行4列（较窄）**：
```
┌───────────────────┬───────────────────┐
│   汇聚部门数      │ 部门汇聚清单表数量│  ← 基础统计（较宽）
│    ● XX个         │    ● XXX张        │
└───────────────────┴───────────────────┘
┌─────────┬─────────┬─────────┬─────────┐
│部门更新率│前置仓更新│链路完成率│链路异常率│  ← 更新率指标（较窄）
│ ● XX.X% │ ● XX.X% │ ● XX.X% │ ● XX.X% │
└─────────┴─────────┴─────────┴─────────┘
```

### 3. 视觉风格 ✅

- **发光圆点**：所有指标使用 `●` 发光圆点代替图标
- **独特颜色**：每个指标有专属颜色和发光效果
- **暗黑科技风格**：半透明背景，发光边框
- **悬停动画**：卡片上浮，圆点放大，发光增强

### 4. Tooltip功能 ✅

**第二行指标卡悬停显示数量/总数**：
- 部门数据更新率：显示 "1,234 / 1,447"
- 前置仓更新率：显示 "1,341 / 1,447"
- 链路完成率：显示 "1,136 / 1,447"
- 链路异常率：显示 "178 / 1,447"

**Tooltip特点**：
- 淡入淡出动画
- 缩放动画（0.9 → 1）
- 对应颜色的边框发光
- 数字千分位格式化

## 计算逻辑

### 基础统计
- **汇聚部门数**：汇总.csv中【部门名称】去重
- **清单表数量**：汇总.csv中【部门表数量】求和

### 更新率指标
- **部门数据更新率**：明细.csv中【部门到前置机状态】为"正常"的比例
- **前置仓更新率**：明细.csv中【前置机到前置仓状态】为"正常"的比例
- **链路完成率**：明细.csv中三个状态**都为"正常"**的比例
- **链路异常率**：明细.csv中**任意一个状态为"异常"**的比例
