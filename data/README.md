# 数据文件说明

本目录包含系统所需的CSV数据文件。

## 文件列表

### 汇总.csv
- **用途**: 包含各部门在不同层级的汇总数据
- **必需字段**:
  - 部门汇聚清单 (或 部门): 部门名称
  - 部门汇聚清单（表数量）: 部门层的表数量
  - 前置机（表数量）: 前置机层的表数量
  - 前置机（数据量）: 前置机层的数据量
  - 前置仓（表数量）: 前置仓层的表数量
  - 前置仓（数据量）: 前置仓层的数据量
  - 贴源层（表数量）: 贴源层的表数量
  - 贴源层（数据量）: 贴源层的数据量

### 明细.csv
- **用途**: 包含每张表的详细对账状态和异常信息
- **必需字段**:
  - 部门: 部门名称
  - 表中文名: 表的中文名称
  - 部门到前置机对账状态: 对账状态
  - 异常原因说明: 异常原因（可选）
  - 前置机到前置仓对账状态: 对账状态
  - 前置仓数据量核对: 核对结果
  - 前置仓到贴源层调度状态: 调度状态

## 数据格式要求

1. **编码**: UTF-8（支持中文字符）
2. **格式**: 标准CSV格式，使用逗号分隔
3. **表头**: 第一行必须包含列名
4. **数值字段**: 表数量和数据量字段应为有效数字
5. **空值处理**: 空值将显示为"-"或"无"

## 数据示例

### 汇总.csv 示例
```csv
部门,部门汇聚清单（表数量）,前置机（表数量）,前置机（数据量）,前置仓（表数量）,前置仓（数据量）,贴源层（表数量）,贴源层（数据量）
交通运输厅,30,28,10000,25,20000,20,15000
教育厅,45,42,15000,40,25000,38,22000
```

### 明细.csv 示例
```csv
部门,表中文名,部门到前置机对账状态,异常原因说明,前置机到前置仓对账状态,异常原因说明,前置仓数据量核对,前置仓到贴源层调度状态,异常原因说明
交通运输厅,业户档案管理表,正常,,正常,,通过,通过,
交通运输厅,车辆信息表,异常,数据缺失,正常,,通过,通过,
```

## 注意事项

1. 确保CSV文件与index.html在同一项目中
2. 使用本地服务器访问页面以避免跨域问题
3. 汇总数据和明细数据的部门名称必须完全匹配
4. 表数量为0的部门将不会在桑基图中显示数据流
